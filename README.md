# CubaCMG - Link x Link

Plataforma de compartir enlaces de WhatsApp con sistema de registro y directorio de usuarios.

## Características

- Sistema de registro e inicio de sesión
- Creación de enlaces de WhatsApp personalizados
- Directorio de usuarios con enlaces activos
- Sistema de likes (un like por usuario)
- Expiración de enlaces en 24 horas
- Personalización con nombre y foto
- Diseño moderno con tema neón púrpura
- Modo claro/oscuro
- Animaciones y efectos visuales

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage para persistencia de datos

## Instalación

1. Clona el repositorio
2. Abre `index.html` en tu navegador

## Despliegue

Este proyecto está diseñado para desplegarse en plataformas como:
- Vercel
- Netlify
- GitHub Pages

## Estructura de Archivos

- `index.html` - Estructura principal de la página
- `style.css` - Estilos y diseño
- `script.js` - Funcionalidad JavaScript
